        \begin{block}{题目大意}
            \begin{itemize}
                \item 累加变量将顺序累加序列，要求任意时刻变量均大于$0$，求其初值。
                \item 关键词：前缀和，签到
                \item 一血：97min
            \end{itemize}
        \end{block}
        \begin{itemize}
            \item 顺序累加具有前缀性，因此使用前缀和计算每个状态。
            \item 若序列均为正数，答案为$1$。
            \item 若序列存在负数，则答案为前缀和中最小值的相反数$+1$。
            \item 对C/C++选手：请注意开long long。
        \end{itemize}
